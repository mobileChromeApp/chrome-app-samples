Getting help or reporting a problem

In addition to these pages, we provide connection information in an online guide. There is also a knowledge base at http://kb.realvnc.com.

If you experience a problem, you can contact Technical Support. We regret that we cannot offer telephone support at this time. Please be sure to include in the body of the enquiry:

  • The version of the Google Chrome web browser.
  • Information about the computer you are sitting in front of: platform and operating system.
  • Information about the computer you are trying to connect to: platform, operating system, and VNC Server (or compatible software) version number.
