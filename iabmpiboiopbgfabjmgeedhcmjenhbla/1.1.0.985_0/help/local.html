Setting up a local connection

You can use VNC Viewer to control a computer connected to the same network as the computer you are sitting in front of, for example over a LAN or VPN. This is typically useful at home or in a small office environment.

Before you can establish a local connection, you must:

  1. Set up the computer you want to control.
  2. Obtain its private network address.

You must perform these operations before you can establish a connection for the first time.
