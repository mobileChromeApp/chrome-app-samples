Using VNC Viewer

Terminology: The client computer is the one you are sitting in front of, running Google Chrome. The host computer is the one you are controlling in the browser window.

What do you want to do?

  • Show the VNC Viewer toolbar
  • Navigate the host computer's desktop
  • Perform certain keyboard operations on the host computer (ie. Ctrl+Alt+Delete)
  • Tweak performance
  • View connection information
  • Disconnect
